# xproc3kit

XProc 3.0 oXygen Framework
  
## How to install it

In oXygen menu bar open Help > Install New Add-ons, into "Show Add-ons from:" paste the URL
  
https://raw.githubusercontent.com/transpect/xproc3kit/main/xproc3.add-on.xml

## What's in it

* editing support
* templates
* validation scenario
